# 🔧 ANTI AI MECHANIC - Lesson 2: Timing Mechanism

## 🎮 Camshaft, Timing Gear and Valve (For Kids)

An interactive educational game where a child learns about the timing mechanism of an internal combustion engine. You need to assemble the timing unit to synchronize the intake and exhaust valves with the piston.

**Live Demo:** https://1-six-sooty-73.vercel.app/lesson2

---

## 📖 Project Description

**"ANTI AI MECHANIC"** is a series of educational mini-games where artificial intelligence "broke" the blueprints, and a real mechanic has to fix everything.

### In the second lesson you need to install:
1. **Camshaft** — controls valve opening and closing
2. **Timing Gear** — transfers rotation from crankshaft
3. **Valve** — lets fuel mixture in and exhaust out

---

## 🎯 Game Goal

Correctly match each timing part with its mounting place. When all three parts are installed, the engine timing is synchronized!

---

## 🕹️ Controls

| Key | Action |
|-----|--------|
| ◀ (Left Arrow) | Move part left |
| ▶ (Right Arrow) | Move part right |
| ▼ (Down Arrow) | Install part |

> On touch devices, on-screen buttons are also available.

## 🚀 How to Play?

1. Open `index.html` in any modern browser
2. Click the **"НАЧАТЬ СБОРКУ ГРМ"** (Start Timing Assembly) button
3. Move the part left/right with arrow keys
4. Press the down arrow to install the part
5. Assemble all 3 parts and watch the timing mechanism work!

**Play online:** https://1-six-sooty-73.vercel.app/lesson2
